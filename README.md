# 💬 WS Chat Client Auth

A real-time chat app with authentication using WebSockets and ReactJS.

## 🔗 Live Demo
[Click Here](https://ws-chat-client-auth.netlify.app/)

## 🚀 Features
- Real-time messaging
- No login required
- Room-based chat
- Works on mobile and desktop

## 📁 Tech Stack
- ReactJS
- WebSocket
- Material UI
